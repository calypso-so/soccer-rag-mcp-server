# Changelog

All notable changes to `@calypsohq/soccer-rag-mcp-server` will be documented in this file.

## Unreleased

### Added

- Added the zero-config World Cup demo mode for `calypso-rag-agent:worldcup`.
- Added npm CLI support through `npx -y @calypsohq/soccer-rag-mcp-server`.
- Added BYOK mode with `CALYPSO_API_KEY` for upload and bucket tools.
- Added stdio smoke coverage for the npm package surface.

### Changed

- Removed non-npm distribution surfaces so npm is the only supported package path.
